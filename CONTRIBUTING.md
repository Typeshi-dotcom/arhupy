# Contributing

Thanks for your interest in contributing to `arhupy`. This project is beginner friendly, and small improvements are very welcome.

## Fork The Repo

1. Open the `arhupy` repository on GitHub.
2. Click **Fork** in the top-right corner.
3. Choose your GitHub account as the destination.

## Clone It Locally

```bash
git clone https://github.com/YOUR-USERNAME/arhupy.git
cd arhupy
```

## Install In Dev Mode

Use editable mode so your local code changes are available immediately:

```bash
pip install -e .
```

## Run Tests

Run the test suite with Python's built-in `unittest` module:

```bash
python -m unittest
```

## Submit A Pull Request

1. Create a new branch for your work.
2. Make your changes.
3. Run `python -m unittest` and make sure everything passes.
4. Commit your changes with a clear message.
5. Push your branch to your fork.
6. Open a pull request and describe what you changed.

Please keep changes focused and include tests when you add or change behavior.
